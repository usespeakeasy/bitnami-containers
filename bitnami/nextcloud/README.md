# Bitnami Secure Image for Nextcloud

## What is Nextcloud?

> Nextcloud is an open-source, self-hosted content collaboration platform that lets you share and sync files, manage calendars and contacts, and communicate via chat and video.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
