# Bitnami Secure Image for GitHub MCP Server

## What is GitHub MCP Server?

> GitHub MCP Server is a Model Context Protocol server that enables AI assistants to interact with GitHub APIs for repository management, issue tracking, pull requests, and more.

[GitHub MCP Server](https://github.com/github/github-mcp-server)

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
