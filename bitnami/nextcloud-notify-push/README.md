# Bitnami Secure Image for Nextcloud Notify Push

## What is Nextcloud Notify Push?

> Nextcloud Notify Push is a high-performance server-sent events push service for Nextcloud, enabling real-time file sync and notifications for clients.

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
